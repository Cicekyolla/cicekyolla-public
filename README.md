# cicekyolla-public

CICEKYOLLA public site (Next.js). Google'ın gördüğü SEO HTML.

- Admin panel + backend: `cicekyolla-api` reposunda (ayrı).
- Bu repo: sadece public site. Mevcut `/api/public/seo/page` JSON endpoint'ini okur.
- Faz 8B-1: SEO iskeleti (title/meta/canonical/robots/h1/body/JSON-LD).
- Faz 8B-2: Figma storefront tasarımı (sonra).

## Env
`.env.example` → Vercel Environment Variables:
- `NEXT_PUBLIC_SITE_URL` — canonical kökü
- `NEXT_PUBLIC_API_ORIGIN` — backend (Render)
